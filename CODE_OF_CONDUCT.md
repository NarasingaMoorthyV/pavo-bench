# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/)
version 2.1 as its code of conduct.

In short: treat everyone who interacts with this project — issues, pull
requests, discussions, email — with respect. Technical disagreement is fine;
personal attacks, harassment, and discrimination are not.

Report violations privately to `moorthyv@sas.upenn.edu`. Reports will be
reviewed and responded to within 7 days.

Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
